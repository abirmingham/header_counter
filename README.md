## Usage
```
header_counter.rb file header [header ...]
```

## Example
```
./count_headers.rb test_data Connection Pragma Cache-Control Accept-Language Accept-Charset Accept-Encoding
```
```
         Header Count
         ------ -----
Accept-Language 38
     Connection 34
  Cache-Control 44
Accept-Encoding 30
 Accept-Charset 34
         Pragma 43
```
